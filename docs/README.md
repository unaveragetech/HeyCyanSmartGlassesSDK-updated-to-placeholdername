# HeyCyan Smart Glasses SDK Documentation

This documentation covers the HeyCyan Smart Glasses SDK functionality in this repository. It includes plans for a new app leveraging these features, including pairing, audio/video capture, streaming, translation, AI image generation and upload, DFU, Wi-Fi transfer, and media management.

- `docs/overview/README.md` - product summary and full feature list.
- `docs/ios/api.md` - iOS SDK public API reference (QCSDK.framework).
- `docs/ios/demo.md` - iOS demo app flow and implementation details.
- `docs/android/api.md` - Android SDK APIs and classes used by the sample.
- `docs/android/demo.md` - Android demo app usage, features and flow.
- `docs/app-architecture/best-route.md` - recommended app architecture and roadmap.
